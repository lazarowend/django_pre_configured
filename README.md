
# Django pré configurado




## Instalação
Clone o repositório
```
git clone "https://github.com/lazarowend/django_pre_configured.git"
```
Crie o ambiente virtual
```bash
# linux
python3 -m venv venv
source venv/bin/activate
```
```bash
# windows
python -m venv venv
venv\Scripts\activate
```

```bash
pip install -r requirements.txt
```
